#### 消息通知

#### GET /api/notice 通知列表

##### 请求参数

| 请求参数名 | 类型   | 说明                            |
| ---------- | ------ | ------------------------------- |
| page      | int   | 页数    默认 1 |
| limit   | int | 条数 10 |


##### 响应参数

| 响应参数名 | 类型   | 说明                            |
| ---------- | ------ | ------------------------------- |
| date      | string   | 日期 eg:'2022-09-21 03:21'      |
| content   | string | 内容 |
| isRead       | int   | 是否已读, 0-未读, 1-已读                          |
| messageId       | int   | 消息ID                        |


#### POST /api/notice/read 消息通知标记已读

##### 请求参数

| 请求参数名 | 类型   | 说明                            |
| ---------- | ------ | ------------------------------- |
| messageId      | int   | 消息ID,多个以逗号隔开. eg: 12,3,4 |


##### 响应参数
无
