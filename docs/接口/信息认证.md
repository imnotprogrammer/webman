#### 信息认证

#### POST /api/verify/stepOne 验证第一步(个人信息)

##### 请求参数

| 请求参数 | 类型   | 说明                            |
| ---------- | ------ | ------------------------------- |
| firstName     | string   | 名字首     |
| middleName    | string | 名字中,可选 |
| lastName      | string   | 名字尾|
| gender      | int   | 性别 1-男 2-女 |
| birth      | string   | 生日 |
| maritalStatus      | string   | 婚姻状态 0-未婚 1-已婚 3-离异 |
| numberOfChildren      | string   | 子女数量 |

##### 响应参数

| 响应参数名 | 类型   | 说明                            |
| ---------- | ------ | ------------------------------- |
| nextStep      | int   | 下一步验证步骤      |
