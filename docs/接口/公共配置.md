#### 配置

#### GET /api/config 配置参数

##### 请求参数

无

##### 响应参数

| 响应参数名 | 类型   | 说明                            |
| ---------- | ------ | ------------------------------- |
| creditAmount      | int   | 初始额度      |
| userType    | int | 用户类型, 0-首贷 1-复贷 |
| canBorrow       | int   | 是否能够申请贷款 0-暂不能申请 1-能申请  |
| verifyStep       | int   | 用户认证第几步, 1,2,3,4 |
| faqs       | obejct   | 常见问题 |
| contacts      | object   | 客服电话 |

| faqs响应参数名 | 类型   | 说明                            |
| ---------- | ------ | ------------------------------- |
| title      | string   | 问题描述      |
| desc    | string  | 问题解答 |

| contacts响应参数名 | 类型   | 说明                            |
| ---------- | ------ | ------------------------------- |
| facebookAccount      | string   | facebook账号     |
| website    | string | 官网地址 |
| email      | string   | 邮箱       |
| tel      | string   | 客服电话 |
