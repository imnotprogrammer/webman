#### 还款

#### POST /api/repay 还款

##### 请求参数
| ---------- | ------ | ------------------------------- |
| orderId      | int   | 订单ID     |
| amount    | int | 还款金额 |
| paymethod    | string | 还款方式 card-银行卡 ussd-电话 va-还款 |

##### 响应参数

| 响应参数名 | 类型   | 说明                            |
| ---------- | ------ | ------------------------------- |
| paymethod      | string   | 还款方式      |
| params   | object |  还款参数 |

#### GET /api/getRepayMethodList 还款方式列表

##### 请求参数
无

##### 响应参数

| 响应参数名 | 类型   | 说明                            |
| ---------- | ------ | ------------------------------- |
| method  | string   | 还款方式,取值ussd,card,va,transfer    |
| name   | string |  还款参数 |
| icon   | string |  图标 |
| params   | object |  参数 |
