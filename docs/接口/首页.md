#### GET /api/home 首页

##### 请求参数

无

##### 响应参数

| 响应参数名 | 类型   | 说明                            |
| ---------- | ------ | ------------------------------- |
| creditAmount      | int   | 初始额度      |
| userType    | int | 用户类型, 0-首贷 1-复贷 |
| canBorrow       | int   | 是否能够申请贷款 0-暂不能申请 1-能申请  |
| verifyStep       | int   | 用户认证第几步, 1,2,3,4 |
