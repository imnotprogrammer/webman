2. 客户端获取到token以后, 根据token 对参数进行签名计算

#### 签名计算

1. 将请求参数进行ksort排序
2. 遍历参数，以'&'拼接字符串.
3. 将第二步得到的字符串，和token进行拼接. 得到str. eg: token+'a=b&c=abc'.
4. 先对拼接得到的字符串进行base64编码,然后进行MD5支持
